About

I'm Eric. I'm a Front-End Web Developer, living in the historical city of Philadelphia.

I create websites from scratch using front-end technologies, learning any language needed in order to turn my vision for a website into reality. Featured in my portfolio you will find some websites which I've completed or remain a work in progress. You can find these projects in my portfolio found below. Each day I work extensively with HTML5, CSS3, and JavaScript, as well as Bootstrap 4 and jQuery with a recent curiosity and inclination towards both React and Node.js. Feel free to contact me via my Linkedin, Github or Twitter.
